# PresenceCV 產品開發藍圖 & 變現規劃 (ROADMAP)

此文件為 PresenceCV 的產品規劃藍圖，旨在協助專案以最快速度驗證商業價值，並逐步整合自動化投遞工具 (104 Job Auto Applier) 實現商業變現。

---

## 🎯 產品變現與收費機制建議 (Monetization & Tiering)

為了在提供良好體驗的同時實現商業化，建議將產品功能劃分為三個層級（免費版、專業版、AI 代理版）：

| 功能項目 | Free (免費版) | Pro (專業版) | Premium Agent (付費代理版) |
| :--- | :---: | :---: | :---: |
| **定價建議** | $0 / 永久 | ☕ $199 NTD / 月 | 🚀 $499 NTD / 月 |
| **履歷 Profile 數量** | 1 個 | 無限制 | 無限制 |
| **AI 履歷解析匯入** | 1 次 / 日 | 無限制 | 無限制 |
| **PDF 匯出** | 基礎單頁 A4 (包含浮水印) | 精美無浮水印 / 支援多頁排版 | 精美無浮水印 / 支援多頁排版 |
| **分享連結** | 快照連結 (靜態一次性) | 快照連結 + Live 實時同步連結 | 快照連結 + Live 實時同步連結 |
| **ATS 關鍵字匹配** | ❌ 無 | 🟢 **完整 AI ATS 深度報告** | 🟢 **完整 AI ATS 深度報告** |
| **104 自動投遞服務** | ❌ 無 | ❌ 無 | 🟢 **AI Agent 自動職缺篩選與自動投遞** |
| **客製化求職信 (Cover Letter)** | ❌ 無 | ❌ 無 | 🟢 **AI 依職缺動態自動撰寫** |

## 🗺️ 開發階段規劃 (Development Phases)

### 📌 階段一：ATS 深度分析實作 (第一優先項目)
*將目前 `InfoEditor.tsx` 中的靜態 Mock 介面升級為真實由 Gemini 驅動的分析工具。*

- [ ] **1. 串接真實 Gemini API 分析目標職缺**
  - 使用者輸入目標職稱或貼上 104 職缺描述 (JD)。
  - 後端 API 調用 Gemini 進行語意分析，對比使用者當前履歷內容。
- [ ] **2. 輸出 ATS 匹配度報告**
  - **匹配分數**：語意契合度評分 (0-100%)。
  - **關鍵字分析**：列出已包含的關鍵字，並標示漏掉的產業關鍵字（如：Product Designer, Figma, Strategic thinking）。
  - **AI 改寫建議**：針對自我介紹 (Summary) 區塊給出具體的「ATS 友善改寫範本」。
- [ ] **3. 串接基礎訂閱制付費閘門 (Paywall)**
  - 串接 Stripe Payment Links 或綠界科技付費連結，實現訂閱解鎖 Pro 會員機制。
  - 於前端限制免費版 Profile 數量與 AI 匯入次數。

---

### 📌 階段二：104 Job Auto Applier & AI Agent 自動投遞整合
*將履歷工具升級為「主動式求職代理 (Active Job Hunting Agent)」，解決重複投遞的痛點。*

- [ ] **1. 自動投遞後端整合 (Puppeteer / Playwright)**
  - 於後端建立基於 Headless 瀏覽器的自動投遞執行器。
  - 串接 104 會員登入（使用加密的 Session Cookie 或帳密環境變數，注重隱私安全）。
- [ ] **2. AI 職缺篩選與求職信 (Cover Letter) 自動化**
  - **自動篩選**：使用者設定求職條件（如：期望薪資、地區、職稱關鍵字），Agent 定時爬取 104 最新職缺，並用 AI 評估該職缺與履歷的契合度，過濾掉不符條件的雜訊。
  - **自動求職信**：針對篩選出的高契合度職缺，AI 自動結合履歷優勢與該職缺 JD，撰寫高度客製化的求職信 (Cover Letter)。
- [ ] **3. 一鍵/全自動投遞控制面板**
  - 前端新增「AI 投遞控制室」頁面：
    - 顯示今日爬取職缺、AI 契合度評分。
    - 提供「手動一鍵投遞」與「全自動代理投遞」開關。
    - 顯示投遞日誌與面試邀請狀態追蹤。

---

### 📌 階段三：多頁列印與進階體驗優化
*提升履歷本體的專業感與進階排版功能。*

- [ ] **1. 多頁 A4 PDF 匯出優化**
  - 解決目前長履歷被強行縮小到單頁 A4 的排版技術債。
  - 支援 `page-break-inside: avoid` 等 CSS 印表機屬性，並提供分頁符號設定。
- [ ] **2. 履歷分享頁面 (ViewPage) 自訂網域**
  - 允許 Premium 用戶將即時履歷連結綁定至自訂網域（例如：`yourname.presencecv.com`）。
